Programação Web de site de decoração.

Iniciar a avaliação do projeto pela pasta ParteDeRick.
Nome do banco de dados: "sousadecor".

Sousa Decor, site para venda de produtos de decoração. Permite aos clientes verem todos os produtos do site, tendo um menu para selecionar a categoria do produto desejado. Possui login e cadastro de vendedores e clientes, e sistema de carrinho, histórico de compras.
